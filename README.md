## Практическая работа №7
Что реализовано:

Создан базовый сервер на Node.js (Express).

Реализована модель пользователя (id, email, first_name, last_name, password) и товара (id, title, category, description, price).

Добавлена регистрация (/api/auth/register) с хешированием пароля.

Добавлен вход (/api/auth/login) с проверкой хеша пароля.

Реализованы CRUD-операции для товаров (/api/products).

Технологии: Node.js, Express, bcrypt.


## Практическая работа №8
Что реализовано:

Доработан сервер из занятия №7.

При успешном входе (/api/auth/login) сервер теперь генерирует и возвращает JWT-токен (access-token).

Создан защищённый маршрут (/api/auth/me), который возвращает информацию о текущем пользователе.

Для проверки токена создан middleware (authMiddleware).

Технологии: Node.js, Express, bcrypt, JWT (библиотека jsonwebtoken).
## Практическая работа №9
Что реализовано:

Доработан сервер из занятия №8.

Введена пара токенов: access (короткоживущий) и refresh (долгоживущий).

Добавлен эндпоинт для обновления токенов (POST /api/auth/refresh).

Реализована "ротация" refresh-токенов (старый удаляется, новый выдается).

Технологии: Node.js, Express, bcrypt, JWT.
