О приложении

Мои заметки — это PWA-приложение для управления заметками с поддержкой офлайн-работы, WebSocket и Push-уведомлений.

Возможности

  • Добавление и удаление заметок
  • Работа в офлайн-режиме
  • Синхронизация через WebSocket
  • Push-уведомления о новых заметках
  • Установка как нативное приложение

Технологии

  • App Shell Architecture
  • Service Worker (Cache First / Network First)
  • Socket.IO для WebSocket
  • Web Push API
  • HTTPS через mkcert
