# Notes App - PWA с WebSocket и Push-уведомлениями

## Функционал
- ✅ App Shell (офлайн-интерфейс)
- ✅ HTTPS (локальный через mkcert)
- ✅ WebSocket (Socket.IO) – синхронизация между вкладками
- ✅ Push-уведомления при добавлении задач
- ✅ Отложенные напоминания с кнопкой "Отложить на 5 минут"

## Технологии
- **Frontend:** HTML/CSS (Chota), JavaScript, PWA
- **Backend:** Node.js, Express, Socket.IO, web-push
- **Безопасность:** самоподписанный HTTPS-сертификат (mkcert)

## Установка и запуск

### 1. Клонировать репозиторий
```bash
git clone <repository-url>
cd Pr17-18
```

### 2. Установить зависимости
```bash
npm install
```

### 3. Сгенерировать VAPID-ключи
```bash
npx web-push generate-vapid-keys
```

### 4. Настроить .env
Создать файл `.env` в корне проекта (на основе `.env.example`) и вставить сгенерированные ключи:
```
VAPID_PUBLIC_KEY=<ваш_public_key>
VAPID_PRIVATE_KEY=<ваш_private_key>
VAPID_SUBJECT=mailto:test@example.com
PORT=3000
```

### 5. Сгенерировать HTTPS-сертификаты
```bash
mkcert -install
mkcert localhost 127.0.0.1 ::1
```

### 6. Запустить сервер
```bash
node server.js
```

### 7. Открыть приложение
```
https://localhost:3000
```

## Тестирование

### Основные проверки
| Функция | Действие | Ожидаемый результат |
|---------|----------|---------------------|
| **App Shell** | Отключить сеть, обновить страницу | Интерфейс загружается (шапка, кнопки) |
| **Навигация** | Нажать "О приложении" | Загружается about.html |
| **Заметки** | Добавить обычную заметку | Сохраняется в localStorage, отображается в списке |
| **WebSocket** | Открыть две вкладки, добавить заметку | Во второй вкладке всплывающее сообщение |
| **Push** | Включить уведомления, добавить заметку | Приходит системное уведомление |
| **Напоминания** | Добавить заметку с напоминанием в будущем | Уведомление приходит в указанное время |
| **Откладывание** | Нажать "Отложить на 5 минут" в уведомлении | Уведомление повторяется через 5 минут |
| **Отписка** | Нажать "Отключить уведомления" | Push не приходят |

### Быстрая проверка напоминаний
1. Добавить напоминание на +1-2 минуты от текущего времени
2. Закрыть вкладку (приложение может быть закрыто)
3. Дождаться — должно прийти push-уведомление с кнопкой "Отложить"
4. Нажать "Отложить на 5 минут"
5. Через 5 минут приходит повторное уведомление

├── .env.example            # Пример .env

## Важные замечания
- Таймеры на сервере **сбрасываются** при перезапуске (допустимо для учебного проекта)
- Время в `datetime-local` — локальное время пользователя
- При изменении `sw.js` инкрементируется `CACHE_NAME`
- VAPID-ключи можно оставить в репозитории — это тестовый проект
